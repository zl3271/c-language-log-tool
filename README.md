 c-language-log-tool
 C语言轻量级日志工具 ## 项目简介 基于开源log.c二次开发，实现极简轻量化C语言日志打印组件，无第三方依赖、兼容C99标准，适配控制台程序调试场景。 ## 技术栈 C语言、文件IO、可变参数、时间处理、宏定义封装 ## 功能实现 
 1. 划分TRACE/DEBUG/INFO/WARN/ERROR/FATAL六级日志等级 
2. 自动打印系统时间、代码文件名、代码行号，方便程序调试
3. 自主新增日志等级开关功能，按需屏蔽低级日志，优化程序输出效率
4. 代码结构规范化，拆分头文件与源文件，遵循软件工程编码规范 ## 编译运行 gcc main.c log.c -o logtool ./logtool
